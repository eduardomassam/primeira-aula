git add . - compacta tudo para ser comitado
git commit -m "mensagem do commit" - comita para uma area temporaria
git push - comita em definitivo no github
git checkout -b develop - para criar uma branch de nome develop
git push origin develop - para postar em outra branch de nome develop

git pull - tras os arquivos para a maquina, se as versoes forem iguais, nao muda nada
git pull origin develop - tras os arquivos para a maquina da branch develop, se as versoes forem iguais, nao muda nada
